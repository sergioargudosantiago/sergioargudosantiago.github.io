# **App Name**: Mi Sitio Personal

## Core Features:

- Navbar: Sticky navigation bar with links: 'Inicio', 'Temario', 'Recursos', and 'Sobre mí'.
- Hero Section: Minimalist introduction with name heading and short professional description.
- Syllabus Section (Temario): Clean, structured list with bold headers for 'Examenes' and a bulleted list for 'Temas'. PDF icon next to each lesson.
- Search Bar UI: Visual search bar in the header or hero section.
- Responsive Design: Optimized layout for both mobile and desktop devices.
- Recursos Section: Organize the 'Recursos' section into 5 main divisions, each representing one of the 5 exams of the 'oposición'. Each division has a title ('Examen 1', 'Examen 2') and a nested list showing the 'Títulos de los temas' corresponding to that exam. Uses a card-based layout or accordion.

## Style Guidelines:

- Main background color: White (#FFFFFF) for a minimalist, clean look.
- Navigation bar and primary headings: Navy Blue (#1E3A8A).
- Accents, buttons, and hover states: Sky Blue (#60A5FA).
- Font: 'Inter' (sans-serif) for body text and headers; a modern, neutral font choice with excellent legibility.
- Plenty of whitespace to ensure readability and a clean user experience.
